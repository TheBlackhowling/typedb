# Version 1.0.9

**Release Date:** 2026-02-02

## Summary

Refactor repository structure by moving examples and integration tests to separate repository to reduce bloat and improve maintainability.

## Key Changes

### Repository Structure
- **BREAKING**: Removed `examples/` directory - moved to [typedb-examples](https://github.com/TheBlackhowling/typedb-examples) repository
- **BREAKING**: Removed `integration_tests/` directory - moved to [typedb-examples](https://github.com/TheBlackhowling/typedb-examples) repository
- Removed example and integration test GitHub Actions workflows - moved to typedb-examples repository
- Updated README.md to reference typedb-examples repository for examples and integration tests

### Impact
- Significantly reduced repository size (removed ~240 files)
- Separated concerns: core library vs examples/testing
- Examples and integration tests now maintained in dedicated repository
- Customers can still access all examples and tests in typedb-examples repository

## Migration Guide

If you were using examples or integration tests from this repository:

1. Examples are now available at: https://github.com/TheBlackhowling/typedb-examples/tree/main/examples
2. Integration tests are now available at: https://github.com/TheBlackhowling/typedb-examples/tree/main/integration_tests
3. All functionality remains the same, just in a different repository

## Detailed Changes

- Deleted `examples/` directory and all subdirectories (postgresql, mysql, sqlite, mssql, oracle, seed)
- Deleted `integration_tests/` directory and all subdirectories (postgresql, mysql, sqlite, mssql, oracle, testhelpers)
- Deleted 10 GitHub Actions workflow files for examples and integration tests
- Updated README.md Examples section to reference typedb-examples repository
- Updated README.md Testing section to reference typedb-examples repository
